Introduction - 

The binance project will be used to trade different kind of ALTcoin.

Please find the steps to execute the project.

Step 1 -: Please execute the DB schema in your local.
Step 2 -: Please run the DepthEventCache.Class which will ask you one altcoin data for the depth event and which will be saved in the db for trading.
Step 3 -: Run the class InitiaterClass.class which will be executing all the trade related data and will notify you if the trade
	  was the profit trade or loss trade.
